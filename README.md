# 🏊 Swiswim

A personal swimming workout tracker — backend API + iOS app (in progress).

## What it does
- Logs swim workouts with sets, distances, and times
- REST API for the iOS client
- Personal project built to scratch my own itch as a swimmer

## Tech stack
**Backend:** Python, FastAPI, PostgreSQL, Docker  
**iOS:** Swift  
**Infrastructure:** Deployed on VPS via docker-compose, nginx
